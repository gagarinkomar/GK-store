GK-store - интернет-магазин аккаунтов и ключей для игр.
Незарегестрированный пользователь может только просмотреть главную страницу.
На главной странице есть сортировка товаров.
При регистрации ты можешь выбрать либо быть продавцом, либо покупателем.
Продавец создает товар, потом он может его редактировать.
Перед тем, как все увидят товар его проверяет админ.
Также админ может создавать промокоды для получения средств на балансе.
После проверки товара админом его могут купить пользователи с ролью покупателя.
После покупок все свои транзакции можно увидеть на странице со списком транзакций.
Также есть отдельные страницы с пользователями, товарами и транзакциями.
Для главных функций взаимодействия с дб есть api.
Для api есть тесты(test.py)


Структура:
app.py - запуск сайта
users.py, transactions.py, promocodes.py, products.py, categories.py - модели дб
user_resources.py, promocode_resources.py, product_resources.py, category_resources.py, transaction_resorces.py - файлы для работы с api
test.py - тесты api
forms.py - формы flask-wtf
api_config.py - файл с api_key и проверкой api_key
